# Poznej & Hraj – Bistro u dvou vikinských přátel

Statická webová prezentace (verze 4) pro projekt "Poznej & Hraj" propojující severské bistro a herní klub.

## Struktura

- `index.html` – základní stránka s informacemi o bistru, programu a rezervačním formulářem.
- `styles.css` – responzivní vzhled s temnou severskou atmosférou.
- `script.js` – jednoduché interakce: mobilní navigace a tlačítko zpět nahoru.

## Lokální spuštění

Otevři soubor `index.html` v libovolném moderním prohlížeči.
